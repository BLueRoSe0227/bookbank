# 이용자 가이드

독서 통장을 처음 쓰시는 분을 위한 안내입니다.

---

## 이 앱은 무엇인가요

**책을 빌릴 때 포인트를 맡기고, 제때 반납하면 돌려받는** 독서 기록 앱입니다.

실제로 책을 주고받는 것이 아니라, 자기가 읽을 책을 등록하고 반납일을 정해
스스로 약속을 지키는 구조입니다. 포인트는 그 약속을 지키게 만드는 장치입니다.

> 💡 포인트는 앱 안에서만 쓰이는 점수입니다. 현금 가치가 없고 환급되지 않습니다.

---

## 시작하기

### 1. 가입

1. 앱에 접속해 **회원가입** 탭을 누릅니다
2. 닉네임(2~20자), 이메일, 비밀번호를 입력합니다
   - 비밀번호는 **영문과 숫자를 모두 포함해 8자 이상**이어야 합니다
3. **가입 신청**을 누릅니다

### 2. 승인 기다리기

가입 즉시 이용할 수 있는 것이 아니라 **관리자의 승인이 필요합니다.**
승인 전에는 "승인 대기 중" 화면만 보입니다.

승인되면 **가입 보너스 500원**이 지급되어 바로 5권까지 빌릴 수 있습니다.

> 승인이 오래 걸린다면 관리자에게 직접 문의해주세요. 자동 알림 메일은 가지 않습니다.

### 3. 앱 설치하기 (선택)

브라우저 주소창 옆의 설치 버튼을 누르면 홈 화면에 아이콘이 생깁니다.
앱처럼 전체 화면으로 쓸 수 있고, 인터넷이 끊겨도 화면은 열립니다.

- **아이폰(Safari)**: 공유 → "홈 화면에 추가"
- **안드로이드(Chrome)**: 메뉴 → "앱 설치"

---

## 포인트 규칙

| 상황 | 변화 | 설명 |
|---|---|---|
| 가입 승인 | **+500원** | 처음 한 번 |
| 대출 신청 | **-100원** | 보증금. 반납하면 돌려받습니다 |
| 정시 반납 | **+100원 +15원** | 보증금 환급 + 보너스 |
| 늦게 반납 | **+100원** | 보증금은 돌려받지만 보너스는 없습니다 |
| 연체 | **하루 -10원** | 반납할 때까지 매일 자정 이후 차감 |
| 대출 연장 | **-50원** | 7일 연장. **돌려받지 못합니다** |
| 서재 방 추가 | **-200원** | 돌려받지 못합니다 |

### 잔액이 뜻하는 것

잔액은 **지금 몇 권까지 빌릴 수 있는가**를 나타냅니다.
보증금이 100원이므로 잔액 500원이면 5권까지 동시에 빌릴 수 있습니다.

홈 화면의 "대출 가능"이 이 숫자를 계산해서 보여줍니다.

> 연체료는 잔액이 0원이 되면 더 이상 차감되지 않습니다. 마이너스가 되지는 않습니다.
> 다만 잔액이 0원이면 새 책을 빌릴 수 없으니, 빌린 책을 반납해 보증금을 되찾아야 합니다.

---

## 책 빌리기

**책 등록** 탭에서 진행합니다.

1. **도서명**을 입력합니다 (필수)
   - 두 글자 이상 입력하면 **이미 등록된 책이 자동으로 제안**됩니다.
     제안을 고르면 저자·출판사·쪽수가 한 번에 채워집니다
   - 화살표 키(↑↓)로 고르고 Enter로 선택할 수 있습니다
2. 저자, 출판사, 장르, 출간일, 쪽수, 줄거리는 선택 사항입니다
   - **쪽수를 입력하면 독서 통계가 훨씬 정확해집니다**
   - 장르는 목록에서 고릅니다 (통계에 쓰입니다)
3. **반납 예정일**을 정합니다 (기본값은 2주 뒤, 최소 내일)
4. **대출 신청**을 누르면 확인 창이 뜹니다
5. 내용을 확인하고 다시 **대출 신청**을 누르면 완료됩니다

> ⚠️ 잔액이 보증금보다 적으면 신청 버튼이 눌리지 않습니다.
> 빌린 책을 반납하면 보증금이 돌아옵니다.

> ⚠️ 같은 책을 이미 빌리고 있으면 중복 신청이 되지 않습니다.

---

## 반납하기

**내 대출** 탭 → 해당 책의 **반납하기** 버튼

1. **별점을 반드시 골라야 합니다** (1~5점)
   - 마우스로 클릭하거나, 키보드 화살표(←→)로도 고를 수 있습니다
2. 감상 메모는 선택 사항입니다 (최대 500자)
   - 여기 적은 메모는 나중에 서재에서 책을 눌렀을 때 다시 볼 수 있습니다
3. **반납**을 누르면 보증금이 환급됩니다

반납일을 넘기지 않았다면 **보너스 15원**이 추가로 붙습니다.

---

## 연장하기

**내 대출** 탭 → **7일 연장 (-50원)** 버튼

- 한 권당 **최대 2회**까지 연장할 수 있습니다
- 연장 수수료 50원은 **돌려받지 못합니다**
- **이미 연체된 책은 연장할 수 없습니다** — 먼저 반납해야 합니다

---

## 나의 서재

완독한 책을 3D 책장에 꽂아 모으는 공간입니다.

### 방 만들기

1. **나의 서재** 탭 → **방 추가** (200원)
2. 방 이름, 설명, 아이콘, 색상을 정합니다
   - 아이콘과 색상은 화살표 키로도 고를 수 있습니다

### 책 꽂기

1. 방을 눌러 들어갑니다
2. **+ 책 꽂기** → 목록에서 책을 고릅니다 → **꽂기**
3. **반납을 마친 책만 꽂을 수 있습니다**

책등의 두께는 쪽수에 비례하고, 색은 제목에 따라 자동으로 정해집니다.
책등을 누르면 별점·완독일·메모를 볼 수 있습니다.

> ⚠️ 방을 삭제하면 꽂아둔 책 정보도 함께 사라집니다.
> 대출 기록 자체는 남으므로 다시 꽂을 수 있습니다.

---

## 포트폴리오 (독서 통계)

완독한 책을 바탕으로 자동 계산됩니다.

- **요약**: 완독 권수, 읽은 쪽수, 하루 평균 쪽수, 평균 별점
- **장르·작가·출판사별** 분포
- **업적 8종**

| 업적 | 조건 |
|---|---|
| 🌱 첫 걸음 | 첫 책 완독 |
| 📚 열 권 돌파 | 10권 완독 |
| 🏆 오십 권 | 50권 완독 |
| ⚡ 속독가 | 하루 평균 50쪽 |
| 🎭 장르 탐험가 | 5개 장르 완독 |
| ⏰ 약속 지킴이 | 연체 없음 |
| 💝 작가 팬 | 한 작가의 책 3권 |
| 🔥 월 3권 | 한 달에 3권 완독 |

> 💡 쪽수를 입력하지 않은 책은 통계에서 제외됩니다. 추정값으로 지표가
> 흐려지지 않게 하기 위해서입니다. 정확한 통계를 원하면 쪽수를 채워주세요.

---

## 독서 통장

포인트가 오간 내역을 통장처럼 볼 수 있습니다. 최근 100건까지 표시됩니다.

날짜 / 항목 / 증감액 / 잔액 순으로 나열되며, 어떤 책 때문에 생긴 거래인지도 함께 보입니다.

---

## 설정

| 기능 | 설명 |
|---|---|
| 닉네임 변경 | 2~20자. 다른 사람과 겹칠 수 없습니다 |
| 비밀번호 변경 | 영문+숫자 8자 이상 |
| 포인트 규칙 보기 | 현재 적용 중인 규칙 확인 |
| **내 데이터 내보내기** | 프로필·대출·거래·서재 전체를 JSON 파일로 저장 |
| 회원 탈퇴 | 아래 참고 |

### 다크 모드

우측 상단의 아이콘으로 밝은 화면 ↔ 어두운 화면을 전환합니다.
설정하지 않으면 기기의 설정을 따릅니다.

### 탈퇴

설정 → **탈퇴하기** → 확인란에 `탈퇴합니다` 를 정확히 입력

> ⚠️ **탈퇴하면 모든 기록이 즉시 삭제되며 복구할 수 없습니다.**
> 탈퇴 전에 "내 데이터 내보내기"로 기록을 저장해두시길 권합니다.

## 자주 묻는 질문

**Q. 비밀번호를 잊어버렸어요.**
로그인 화면의 "비밀번호를 잊으셨나요?"를 누르고 이메일을 입력하면 재설정 링크가 갑니다.
보안상 가입 여부와 무관하게 같은 안내가 표시되니, 메일이 오지 않으면 가입한 이메일이 맞는지 확인해주세요.

**Q. 알림 메일이나 푸시 알림이 오나요?**
푸시 알림은 없습니다. 반납이 임박하면 앱 홈 화면 상단에 배너로 알려드립니다.

**Q. 인터넷이 없어도 쓸 수 있나요?**
화면은 열리지만 데이터를 불러오거나 저장할 수는 없습니다. 조회·등록·반납은 모두 인터넷이 필요합니다.

**Q. 서버에 연결할 수 없다고 나옵니다.**
잠시 후 다시 시도해주세요. 계속된다면 관리자에게 알려주세요.
(무료 서비스 특성상 오래 사용이 없으면 일시적으로 멈출 수 있습니다.)

**Q. 실수로 잘못된 책 정보를 등록했어요.**
등록된 도서 정보는 여러 사람이 함께 쓰기 때문에 직접 고칠 수 없습니다. 관리자에게 문의해주세요.

**Q. 연체료가 계속 쌓이나요?**
반납할 때까지 하루 10원씩 쌓입니다. 다만 잔액이 0원이 되면 더 이상 차감되지 않습니다.

**Q. 포인트를 현금으로 바꿀 수 있나요?**
아니요. 앱 안에서만 쓰이는 점수이며 환급·양도되지 않습니다.
