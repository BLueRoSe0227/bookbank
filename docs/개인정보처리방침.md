# 개인정보처리방침

> ⚠️ **작성 중인 초안입니다.**
> 수집 항목·보유 기간 등 사실 관계는 실제 구현을 확인해 채웠습니다.
> **[대괄호로 표시된 부분]** 은 운영자가 직접 기입해야 합니다.
> 실제 서비스 공개 전에는 법률 전문가의 검토를 받으시길 권합니다.

최종 수정일: 2026-07-20

## 1. 개인정보의 수집 항목 및 방법

본 서비스(이하 "독서 통장")는 다음 정보를 수집합니다.

| 항목 | 수집 시점 | 필수 여부 | 목적 |
|---|---|---|---|
| 이메일 주소 | 회원가입 | 필수 | 로그인 식별, 비밀번호 재설정 |
| 비밀번호 | 회원가입 | 필수 | 인증 (단방향 암호화되어 저장) |
| 닉네임 | 회원가입 | 필수 | 서비스 내 표시 이름 |
| 도서 정보·독서 기록 | 서비스 이용 중 | 선택 | 대출 관리, 독서 통계 제공 |
| 포인트 거래 내역 | 서비스 이용 중 | 자동 생성 | 잔액 계산, 내역 조회 |
| 서비스 이용 기록 | 서비스 이용 중 | 자동 생성 | 서비스 개선 (대출·반납·연장 발생 여부) |

인증 정보는 Supabase Auth 가 관리하며, 비밀번호는 운영자도 열람할 수 없습니다.

## 2. 개인정보의 보유 및 이용 기간

- 회원 탈퇴 시 즉시 파기합니다. 설정 화면의 "탈퇴하기"로 직접 탈퇴할 수 있습니다.
- 탈퇴 시 프로필·대출 기록·거래 내역·서재 정보가 함께 삭제됩니다.
- 도서 정보(제목·저자 등)는 개인을 식별하지 않는 공용 데이터이므로 삭제되지 않습니다.
- 관련 법령에 따라 보존이 필요한 경우 해당 기간 동안 보관합니다.

## 3. 개인정보의 제3자 제공

운영자는 이용자의 개인정보를 제3자에게 제공하지 않습니다.

## 4. 개인정보 처리의 위탁

서비스 운영을 위해 아래 업체에 개인정보 처리를 위탁하고 있습니다.

| 수탁자 | 위탁 업무 | 보관 위치 |
|---|---|---|
| Supabase Inc. | 데이터베이스, 인증, 메일 발송 | **[리전 확인 후 기입 — 예: 서울(ap-northeast-2)]** |
| Cloudflare, Inc. | 웹사이트 호스팅 | 글로벌 CDN |

## 5. 이용자의 권리

이용자는 언제든지 다음 권리를 행사할 수 있습니다.

- **열람**: 설정 → "내 데이터 내보내기"로 보유 중인 모든 정보를 JSON 으로 내려받을 수 있습니다.
- **정정**: 설정 화면에서 닉네임·비밀번호를 직접 변경할 수 있습니다.
- **삭제**: 설정 → "탈퇴하기"로 모든 정보를 삭제할 수 있습니다.
- 그 밖의 요청은 아래 연락처로 문의해주세요.

## 6. 개인정보의 안전성 확보 조치

- 모든 통신은 HTTPS 로 암호화됩니다.
- 데이터베이스 접근은 행 수준 보안(RLS)으로 통제되어, 이용자는 자신의 데이터만 조회할 수 있습니다.
- 비밀번호는 복호화가 불가능한 형태로 저장됩니다.
- 포인트 잔액과 권한은 이용자가 직접 수정할 수 없도록 데이터베이스 권한으로 차단되어 있습니다.
